# slot-game-project
Tracking my C++ progress by designing and implementing a slot game step by step.
## Goals
- Improve C++ fundamentals through hands-on development
- Build slot game logic step by step
- Explore simulations, probability, and balancing
- Practice structuring larger codebases

## Project Approach
This project is being developed incrementally.
Each stage adds more complexity, moving from basic mechanics to more complete architecture and simulation systems.

## Planned Areas
- Core reel/spin logic
- Symbol definitions and paylines
- Win evaluation
- Random outcome generation
- RTP / probability simulations
- Config-driven game setup
- Separation of game logic, simulation, and architecture layers

## Progress
- [x] Basic spin system
- [x] Symbol and reel model
- [x] Payline evaluation
- [x] Payout system
- [x] Simulation tooling
- [ ] Refactoring into cleaner architecture
- [ ] Advanced features

## Why this repo exists
This is primarily a personal learning project, not a production game.
The goal is to use one evolving project to learn C++ more deeply over time.
